# tools

Developer tools and migration scripts for pushup-stats-service.

## Scripts

### Import user configs to Firestore Emulator

Imports user configs from a JSON export into the local Firestore Emulator:

```bash
nx run tools:import-user-configs
```

Requires the Firestore emulator running at `localhost:8080` and a `data/user-config-export.json` file (generated by the export script below).

### Export NeDB user configs

Exports all user configs from the local NeDB database to JSON:

```bash
nx run tools:export-nedb-user-configs
```

Writes to `data/user-config-export.json`.
